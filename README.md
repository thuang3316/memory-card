# Memory Card Game

The game rule is simple. You click 10 different images in a role, you win! Any repeated click makes you lose.

I use my favorite anime characters from [BanG Dream! It's MyGO!!!!!](https://en.wikipedia.org/wiki/BanG_Dream!_It%27s_MyGO!!!!!).

There is also a little surprise if you win the game, try to uncover it yourself :)

Here is the [demo](https://memory-card-tau-self.vercel.app/), check it out!
